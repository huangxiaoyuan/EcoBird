准备文件:
在当前目录下创建一个文件夹 my_docs。
把你的 PDF 文件（如 2023_tc.pdf）放入 my_docs 文件夹中。
确保 server.py 和 client.py 放在同一级目录。

启动服务器:
打开一个终端窗口，运行：
python server.py
第一次运行时，它会构建知识库并在当前目录生成 chroma_db 文件夹。
以后再运行时，它会直接加载这个文件夹，速度很快。

启动客户端:
打开另一个终端窗口，运行：
python client.py

操作:
在客户端输入问题：“澳门的黑脸琵鹭数量”。
如果更新了 PDF 文件，在客户端输入 /rebuild，服务器会重新处理所有文档。
